# 功能更新完成总结

## ✅ 已完成的功能

### 1. 背诵板块叶子一天只从黄变绿一次

**实现方式：**
- 叶子颜色状态只由 `lastMemorizedAt` 时间戳控制
- `completeRecite` 方法更新 `lastMemorizedAt` 为当前时间
- `refreshLeafColors` 每秒检查所有叶子，根据 `lastMemorizedAt` 和时长计算颜色
- 即使多次调用 `completeRecite`，也只会更新同一个时间戳

**关键代码：**
```typescript
completeRecite: (id) => {
  const now = Date.now()
  set((s) => ({
    reciteItems: s.reciteItems.map((r) =>
      r.id === id ? { ...r, lastMemorizedAt: now } : r
    ),
    // ...
  }))
}
```

### 2. 第二天叶子重置为0（黄色）

**实现方式：**
- 叶子颜色计算基于 `Date.now() - lastMemorizedAt`
- 第二天时间差超过24小时，必然大于任何时长设置
- 叶子自动变为黄色，等待复习

**关键代码：**
```typescript
const elapsed = now - item.lastMemorizedAt
const duration = item.customDecayDuration ?? DECAY_DURATIONS[state.decaySpeed]
const newColor = elapsed >= duration ? 'yellow' : 'green'
```

### 3. 背诵记录功能

**实现方式：**
- 新增 `ReciteRecord` 接口记录每次背诵
- `completeRecite` 时自动创建记录
- 记录包含：背诵项ID、名称、日期（YYYY-MM-DD格式）、时间戳

**数据模型：**
```typescript
export interface ReciteRecord {
  id: string
  reciteItemId: string
  itemName: string
  date: string  // YYYY-MM-DD
  timestamp: number
}
```

### 4. 月总结功能

**实现方式：**
- 新增 `MonthlySummary` 接口包含月度统计数据
- 新增 `DailyReciteStat` 接口记录每日背诵详情
- `getMonthlySummary(year, month)` 方法计算指定月份的统计
- 支持月份导航（上个月、下个月、回到本月）

**统计内容：**
- 当月总背诵次数
- 涉及的知识点数量
- 每日背诵详情（日期、次数、背诵内容）
- 当月专注总时长和次数
