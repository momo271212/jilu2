## 🔐 安全提示

⚠️ **重要**：你的 GitHub Token 已经配置到远程仓库 URL 中

**建议操作：**

1. **重新生成 Token**（更安全）
   ```bash
   # 访问 https://github.com/settings/tokens
   # 删除旧的 token: ghp_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
   # 生成新的 token
   ```

2. **使用 GitHub Secrets**（如果有多人协作）
   - 不要在代码中硬编码 token
   - 使用 GitHub Actions Secrets

3. **定期轮换 Token**
   - 建议每 90 天更换一次
   - 如果怀疑泄露，立即删除并重新生成
